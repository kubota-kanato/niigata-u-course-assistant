# 機能1：単位修得状況の自動判定

## Step 1：読み込み＋集計
```python
import openpyxl, unicodedata
from collections import defaultdict

def norm(s):
    s = unicodedata.normalize('NFKC', str(s).strip()) if s else ''
    return s.rstrip('◆★※◇△▲▽▼')

wb = openpyxl.load_workbook('単位修得状況.xlsx')
ws = wb['単位修得状況']
student_name = ws.cell(1,2).value
student_id = ws.cell(1,4).value
affiliation = ws.cell(2,2).value
grade = ws.cell(2,4).value
enrollment_year = 2000 + int(student_id[1:3])

# Row6〜データ: D=開講番号 E=科目 G=科目区分 H=必選区分 I=単位数 L=合否
subject_records = defaultdict(list)
for row in ws.iter_rows(min_row=6, values_only=True):
    no,field,level,code,subject,teacher,category,req_type,credits,score,grade_val,pass_fail,gp,year,semester,*rest = row
    if code is None: continue
    subject_records[norm(subject)].append({
        'subject':subject,'category':category,'req_type':req_type,
        'credits':credits,'pass_fail':pass_fail,'year':year,'semester':semester})

# 再履修対応：同一科目名に「合」が1つでもあれば合格扱い
passed = {}; passed_subjects = set(); failed = []
for sn, recs in subject_records.items():
    if any(r['pass_fail']=='合' for r in recs):
        passed_subjects.add(sn)
        best = [r for r in recs if r['pass_fail']=='合'][-1]
        passed[best['category']] = passed.get(best['category'],0) + best['credits']
    elif recs[-1]['pass_fail']=='否':
        r = recs[-1]
        failed.append({'subject':r['subject'],'category':r['category'],'req_type':r['req_type'],'credits':r['credits']})
```

## Step 2：未修得必修科目の特定
program_requirements.jsonの該当プログラムから●マーク科目を取得し、**norm()で正規化してからpassed_subjectsと比較**。一致しないものが未修得。
⚠️ Excelは全角（Ａ Ｂ Ⅰ）、JSONは半角（A B I）。必ずnorm()で正規化比較すること。

## Step 3：要件照合
**教養教育**：合計21単位以上が必須。小区分別の下限（健康スポーツ2単位等）はgraduation_rules.jsonに記載があるが、**合計21単位の充足だけを判定する。小区分の不足は表示しない。**

科目区分マッピング：
- 教養：英語/初修外国語/健康・スポーツ/情報リテラシー/新潟大学個性化科目/人文社会・教育科学/自然科学/医歯学/大学学習法/留学生基本科目
- 専門：大学学習法/自然系共通専門基礎(各分野)/理学基礎演習/基礎実習科目/共通ベーシック科目/主専攻コア科目/主専攻専門科目/他主専攻コア・専門科目

注意：「大学学習法」は教養と専門の両方にある。専門側は理学スタディ・スキルズ（2単位必修）。

**専門教育**：84単位以上が必須。

**自然系共通専門基礎**：
- 必須判定は2つだけ：(1)合計12単位以上 (2)1分野が6単位を超えていないこと
- ⚠️ **分野別の最低単位数（min_credits）は判定に使わない。表示も不要。** 物理学0単位でも合計12以上＋各分野6以下なら「充足」。

```
・「○○の科目についてもっと詳しく教えて」→ 個別の科目について詳しく説明します
```

# 機能2：時間割案の生成

対象学期と好み（バランス型/集中型/ゆとり型）を質問。

courses_2025.csvの構造：開講番号,分野1〜3,水準,科目名,単位,開講時期,代表教員,所属,定員,主聴講対象,遠隔,曜日,時限,授業形態

科目選定優先度：(1)未修得必修 (2)再履修 (3)選択必修 (4)自然系共通不足分野 (5)教養不足区分 (6)選択

制約：CAP制28単位以内／時限重複なし／ターム正しく配置

3案生成：A=バランス型　B=集中型（空き日あり）　C=ゆとり型
