# 新大理学部 履修アシスタント

新潟大学理学部の履修相談に特化した ChatGPT GPTs です。
学務情報システムからダウンロードした Excel をアップロードするだけで、卒業要件との差分を自動判定します。

## 機能

1. **単位修得状況の自動判定** — 教養・専門・合計の充足状況を即座に計算
2. **時間割案の自動生成** — 不足科目を優先配置した 3 パターンを画像で出力
3. **教員免許の取得判定** — 数学・理科 × 中学・高校の 4 パターンに対応
4. **判定根拠の説明** — なぜその判定になったかをステップごとに表示

## 対応範囲

- 理学部理学科 7 プログラム（数学・物理学・化学・生物学・地質科学・自然環境科学・フィールド科学人材育成）
- 2023〜2025 年度入学者
- 専門力 / 総合力トラック

## ファイル構成

```
system_prompt.txt          # GPTs のシステムプロンプト（v2.5）
knowledge/
  graduation_rules.json    # 卒業・進級要件（14.9 KB）
  program_requirements.json # プログラム別科目要件（28.6 KB）
  teacher_license.json     # 教員免許取得要件（14.7 KB）
  courses_2025.csv         # 2025 年度開講科目（134 KB）
```

## 技術的なポイント

- **PDF ゼロ**: 履修の手引き 100 ページ超を全て JSON に構造化。OCR 依存を排除
- **Unicode 正規化**: `unicodedata.normalize('NFKC')` + 末尾記号除去で全角半角の不一致に対応
- **再履修ロジック**: 同一科目を科目名でグルーピングし、合格が 1 つでもあれば合格扱い
- **ASCII 時間割**: Code Interpreter に日本語フォントがないため、matplotlib で ASCII のみの画像を生成
